# Story 2.6: Client Detail View (v3 — Session Timeline)

## Status
Not Started

## Story

**As a** executive coach,
**I want** to see my client's complete session history, coaching goal, and pending action items in one view,
**so that** I can quickly understand where we are in our coaching journey before any session.

## Acceptance Criteria

1. Client detail page exists at `/dashboard/clients/[id]`
2. **Header section** displays: client name (h1), goal (secondary text below name), "Coaching since [start_date]" (if set), stats row: Total Sessions | Pending Actions | Last Session date
3. **Session Timeline** (main content): reverse-chronological list of sessions; "Upload Session Transcript" button at top; empty state with upload CTA when no sessions exist
4. Each session card shows (collapsed): date, title, summary snippet (first 150 chars), key topic badges (max 3), action item count badge
5. Session cards expand on click to show: full summary, all key topics, full action item list, "View Transcript" link
6. Sessions with `status = 'processing'` show a spinner/progress indicator
7. **Pending Actions section**: lists all incomplete action items for this client; quick-complete checkbox; grouped or flat list
8. **"Ask Solis about [client name]"** button visible; opens SolisPanel scoped to this client
9. Loading skeleton displayed while data fetches
10. Mobile responsive layout
11. Back navigation to `/dashboard/clients`
