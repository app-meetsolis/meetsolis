# Story 3.4: AI Summary Generation

## Status
Not Started

## Story

**As a** executive coach,
**I want** an AI-generated summary of my coaching session with key topics and action items extracted automatically,
**so that** I spend zero time on post-session documentation.

## Acceptance Criteria

1. `POST /api/sessions/[id]/summarize` triggers AI summarization from session transcript text
2. AI generates: title (if not already set), summary paragraph, key_topics[] array, action_items[] (description + assigned_to)
3. Action items auto-created as `action_items` records with `assigned_to: 'coach' | 'client'`
4. Session embedding generated from summary text → stored in `sessions.embedding` (vector 1536)
5. Session `status` → `'complete'` on success, `'error'` on failure
6. `AI_PROVIDER` env var controls which provider is used (placeholder | claude | openai)
7. Placeholder provider returns instant mock summary (no API cost, for local dev)
8. Manual edit of generated summary supported via `PUT /api/sessions/[id]`
9. "Regenerate Summary" option available (re-calls summarize endpoint)
10. Cost target: <$0.05/session (Claude with prompt caching)
11. Uses ICF-compliant prompt templates (coaching vocabulary, not clinical)

## Provider Abstraction

```typescript
// apps/web/src/lib/ai/provider.ts
interface AIProvider {
  summarizeSession(transcript: string, ctx: ClientContext): Promise<SessionSummary>
  generateEmbedding(text: string): Promise<number[]>
  queryIntelligence(query: string, sessions: SessionContext[], clientCtx?: ClientContext): Promise<SolisResponse>
}

interface SessionSummary {
  title: string
  summary: string
  key_topics: string[]
  action_items: { description: string; assigned_to: 'coach' | 'client' }[]
}

interface ClientContext {
  name: string
  goal?: string
  coaching_since?: string
}
```

## Dev Notes

- Install `@anthropic-ai/sdk` and `openai` packages
- Placeholder provider should return realistic-looking coaching session output for good DX
- Claude prompt caching: use `cache_control: { type: 'ephemeral' }` on the system prompt to cache ICF template across requests
- For structured output: use Claude's native structured output or instruct JSON response with clear schema in prompt
