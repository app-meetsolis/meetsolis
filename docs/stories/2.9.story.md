# Story 2.9: Navigation Refactor (Left Sidebar)

## Status
Done

## Story

**As a** executive coach,
**I want** a clean left sidebar navigation with clear sections,
**so that** I can move quickly between my clients, Solis Intelligence, and settings.

## Story Context

**Existing System Integration:**
- Replaces: `apps/web/src/components/dashboard/Navigation.tsx` (horizontal top-nav, to be deleted after)
- Replaces: header block in `apps/web/src/app/(dashboard)/layout.tsx`
- Reuse pattern from: `Navigation.tsx` — `usePathname()` for active state, `useRouter()`, Lucide icons
- Touch points: all `(dashboard)` pages share this layout; no page-level changes needed
- User display: `apps/web/src/components/dashboard/UserProfile.tsx` exists — reuse or inline via `useUser()` from `@clerk/nextjs`
- Technology: Next.js App Router, Tailwind, Shadcn UI, Lucide React, Clerk

## Acceptance Criteria

1. Left sidebar rendered on all dashboard pages (`/dashboard/*`)
2. **Navigation items (exact):**
   - **Clients** → `/clients` (icon: `Users`)
   - **Intelligence** → `/intelligence` (icon: `Sparkles`)
   - **Settings** → `/settings` (icon: `Settings`)
3. Active item visually highlighted — use `bg-[#E8E4DD]` or left border accent; detect via `usePathname()`
4. Sidebar width: 200–220px on desktop (`w-52` or `w-56`)
5. Sidebar collapses to hamburger on mobile (< 768px / `md:` breakpoint)
6. Mobile: compact top bar with hamburger → opens full-height overlay sidebar
7. Mobile: overlay closes on nav item click or backdrop click
8. User avatar + name at sidebar bottom; "Sign out" button accessible
9. Old top header (sticky header with `Navigation` + `UserProfile`) removed from layout
10. Main content fills remaining width to the right of sidebar (`flex-1 overflow-auto`)
11. Existing clients page renders correctly inside new layout (no visual regression)
12. Placeholder page at `/intelligence` returns 200 (not 404)
13. Placeholder page at `/settings` returns 200 (not 404)

```tsx

export default function DashboardLayout({ children }) {
  const [isMobileOpen, setIsMobileOpen] = useState(false);
  return (
    <>
      <UserTracking />
      <ErrorBoundary>
        <div className="flex h-screen bg-[#E8E4DD]">
          <LeftSidebar isMobileOpen={isMobileOpen} onClose={() => setIsMobileOpen(false)} />
          {/* Mobile top bar */}
          <div className="flex flex-1 flex-col md:hidden">
            <div className="flex h-12 items-center border-b bg-white px-4">
              <button onClick={() => setIsMobileOpen(true)}>☰</button>
              <span className="ml-3 font-bold text-[#001F3F]">MeetSolis</span>
            </div>
            <main className="flex-1 overflow-auto">{children}</main>
          </div>
          <main className="hidden flex-1 overflow-auto md:block">{children}</main>
        </div>
      </ErrorBoundary>
    </>
  );
}
```

**Active state detection:**
