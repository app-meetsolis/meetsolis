# Plans Archive

All planning session outputs saved here before execution.

## Structure
```
plans/
└── 2026-03-10/
    ├── 14-32-client-detail-view.md
    └── 16-05-auth-refactor.md
└── 2026-03-11/
    └── 09-15-session-recording.md
```

## Naming Convention
`plans/{YYYY-MM-DD}/{HH-MM}-{title}.md`

- Date folder = day the plan was created
- File name = time (HH-MM) + short title (kebab-case)

## Process
1. Finish planning session
2. Save plan here before opening execution session
3. Reference it in execution session: "plan is at plans/2026-03-10/14-32-client-detail-view.md"
